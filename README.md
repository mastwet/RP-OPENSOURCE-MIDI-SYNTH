# RP-OPENSOURCE-MIDI-SYNTH

> **Note:** The RapidPulze open-source organization has been dissolved due to internal member conflicts and inconsistent interests. This repository is an archive of open-source MIDI instrument projects completed by me.

## Introduction

This project is a collection of open-source musical instrument projects based on the MIDI protocol, developed using embedded platforms such as ESP32. These projects were initially initiated by the RapidPulze open-source organization and, after its dissolution, have been maintained and archived here by me.

## Project List

### Chocoly
MIDI music controller project.

### Oreoly
MIDI sequencer and controller.

### RapidPulze-BuleBottle
MIDI instrument based on the "Blue Bottle" concept.

### RP-0x33
MIDI synthesizer module with specific designation.

### RP-0x88
MIDI controller/synthesizer with specific designation.

### RP-66Channel
Multi-channel MIDI audio processor.

### RP-UI
Universal user interface module.
