# RP-OPENSOURCE-MIDI-SYNTH

> **Note:** RapidPulze开源组织因内部成员斗争和利益不一致问题已解散。本仓库是本人完成的开源MIDI乐器合集存档。

## 简介

本项目是一系列基于MIDI协议的开源音乐乐器项目，使用ESP32等嵌入式平台开发。这些项目最初由RapidPulze开源组织发起，在组织解散后由本人继续维护并在此归档。

## 项目列表

### Chocoly
MIDI音乐控制器项目。

### Oreoly
MIDI音序器和控制器。

### RapidPulze-BuleBottle
基于"蓝瓶"概念的MIDI乐器。

### RP-0x33
特定编号的MIDI合成器模块。

### RP-0x88
特定编号的MIDI控制器/合成器。

### RP-66Channel
多通道MIDI音频处理器。

### RP-UI
通用用户界面模块。
